# HexTools для IDA Pro
**Плагин для IDA Pro для копирования и патчинга байт в виде HEX прямо из окна дизассемблера.**

Форк [HexCopy от OALabs](https://github.com/OALabs/hexcopy-ida), расширенный функцией **Paste Hex** — патч байт по адресу курсора через вставку hex строки из буфера обмена или ручной ввод.

## 📖 О проекте

![How to Use](https://github.com/mcsheld/IDA-HexTools/blob/main/Images/HexTools.gif?raw=true)

HexTools добавляет в окно дизассемблера IDA Pro два взаимодополняющих действия: копирование выделенных байт в виде hex строки и обратный патч байт по текущему адресу. Если буфер обмена не содержит корректных hex данных — автоматически открывается диалог ввода.

### Возможности

| Функция | Горячая клавиша | Описание |
|---|---|---|
| 📋 **Copy Hex** | `Ctrl+H` | Копировать выделенные байты в буфер как hex строку в верхнем регистре |
| 📥 **Paste Hex** | `Ctrl+Shift+H` | Патч байт по адресу курсора из буфера или ручного ввода |
| 🖱️ **Контекстное меню** | ПКМ → Hex Tools | Оба действия доступны в popup меню дизассемблера |
| 🔍 **Умный fallback** | — | Если ничего не выделено — копируется элемент под курсором |
| 📝 **Диалог ввода** | — | Открывается автоматически, если в буфере нет корректных hex данных |
| 🔄 **Авто-переход** | — | Курсор перемещается за последний запатченный байт (настраивается) |
| 🪟 **Лог в Output** | — | Детали копирования/вставки выводятся в окно Output IDA (настраивается) |

### Поддерживаемые форматы hex для вставки

Парсер принимает большинство распространённых форматов:

```
A1 B2 C3 D4          через пробел
A1:B2:C3:D4          через двоеточие
A1-B2-C3-D4          через дефис
A1B2C3D4             чистая hex строка
0xA1 0xB2 0xC3       с префиксом 0x
0xA1,0xB2,0xC3       с префиксом 0x через запятую
```

## 🚀 Быстрый старт

### Системные требования

- **IDA Pro** 9.2 или выше
- **Python** 3.x
- **PySide6** — поставляется в комплекте с IDA Pro 9.x

### Установка

1. Скачайте [`HexTools.py`](./HexTools.py)
2. Скопируйте файл в директорию плагинов IDA:

```
# Windows
%APPDATA%\Hex-Rays\IDA Pro\plugins\

# Linux / macOS
~/.idapro/plugins/
```

3. Перезапустите IDA Pro — плагин загрузится автоматически

## ⚙️ Настройка

В начале файла `HexTools.py` есть параметры, которые можно изменить:

```python
# Переместить курсор на байт после последнего вставленного
JUMP_AFTER_PASTE = True

# Выводить информацию о копировании/вставки в окно Output IDA
OUTPUT_COPY_INFO = True
OUTPUT_PAST_INFO = True
```

## 🗂️ Использование

**Копирование (Copy Hex)**
1. Выделите байты в окне дизассемблера
2. Нажмите `Ctrl+H` или ПКМ → **Hex Tools → Copy Hex**
3. Hex строка скопирована в буфер и выведена в окно Output

**Вставка (Paste Hex)**
1. Скопируйте hex строку в буфер обмена из любого источника
2. Установите курсор на нужный адрес в окне дизассемблера
3. Нажмите `Ctrl+Shift+H` или ПКМ → **Hex Tools → Paste Hex**
4. Если в буфере корректные hex данные — байты патчатся немедленно
5. Если нет — откроется диалог для ручного ввода
